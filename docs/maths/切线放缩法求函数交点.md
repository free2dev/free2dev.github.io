这是一篇关于利用切线放缩法（Tangent Line Relaxation）解决函数与水平直线交点问题（特别是横坐标之差/和的范围估计）的数学专题文章。

------

# 切线放缩法求函数交点

摘要

在函数综合问题中，求解超越函数 $f(x)$ 与水平直线 $y=k$ 的交点（即方程 $f(x)=k$ 的根）往往无法得到精确的代数解。本文探讨如何利用**切线放缩（Tangent Line Relaxation）**的几何意义，通过构造特征点处的切线不等式，将复杂的曲线问题转化为线性问题，从而对交点横坐标的差值（弦长）或和值进行精确的范围估计。

------

### 一、 核心理论与几何背景

切线放缩的核心思想源于函数的**凹凸性（Convexity/Concavity）**。

对于二阶可导函数 $f(x)$：

1. 若 $f''(x) > 0$（函数下凸），图象位于任意切线的上方（Except the tangent point）。
   - **不等式：** $f(x) \ge f(x_0) + f'(x_0)(x - x_0)$
2. 若 $f''(x) < 0$（函数上凸），图象位于任意切线的下方。
   - **不等式：** $f(x) \le f(x_0) + f'(x_0)(x - x_0)$

**常见的切线放缩模型：**

- **指数模型：** $y = e^x$ 在 $x=0$ 处的切线为 $y=x+1$，故 $e^x \ge x+1$。
- **对数模型：** $y = \ln x$ 在 $x=1$ 处的切线为 $y=x-1$，故 $\ln x \le x-1$。

------

### 二、 问题模型：水平截线与“根的区间”

问题描述：

设单峰函数 $f(x)$（先增后减或先减后增）与水平直线 $y=k$ 有两个交点 $A(x_1, k)$ 和 $B(x_2, k)$，且 $x_1 < x_2$。我们需要估计 $|x_2 - x_1|$（根的差）或 $x_1 + x_2$（根的和）的取值范围。

**难点：** $x_1, x_2$ 是超越方程的根，无法直接写出表达式。

切线放缩法的策略：

利用曲线的切线来“包裹”或“支撑”曲线，切线与直线 $y=k$ 的交点往往比曲线的交点更容易计算。通过比较切线交点与实际根的位置，从而获得不等式链。

------

### 三、 图解：利用切线放缩估计根之差 $|x_2 - x_1|$

假设 $f(x)$ 为**上凸函数**（如 $f(x) = \ln x - x$ 或 $f(x) = -x^2$），且存在极大值点 $x_0$。直线 $y=k$ 与曲线交于 $x_1, x_2$。

#### 1. 构造外围切线（Outer Bounds）

我们在 $x_1$ 左侧取一点 $a$（通常是定义域端点或特殊点），在 $x_2$ 右侧取一点 $b$，分别作切线 $L_1, L_2$。

由于函数上凸，曲线在切线下方。因此，切线 $L_1, L_2$ 与直线 $y=k$ 的交点（记为 $\hat{x}_1, \hat{x}_2$）通常会将实际的根 $x_1, x_2$ “夹”在中间（或在某些特定构型下提供单侧界限）。

#### 2. 构造极值点切线（Flat Top）

虽然极值点处的切线是水平的（$f'(x_0)=0$），不能直接用于解 $x$，但它提供了高度的上限。更常用的是**割线（Secant Line）**与切线结合使用。

#### 3. 典型应用案例分析

**案例：估计 $f(x) = x e^{-x}$ 的根之差**

设 $f(x) = x e^{-x}$，其在 $(1, e^{-1})$ 处取得极大值。

考虑方程 $x e^{-x} = k$ ($0 < k < e^{-1}$) 的两个根 $x_1, x_2$ ($x_1 < 1 < x_2$)。求证 $x_2 - x_1$ 的某种界限。

步骤一：代数放缩转化



$$x_1 e^{-x_1} = x_2 e^{-x_2} = k$$



两边取对数：$\ln x_1 - x_1 = \ln x_2 - x_2 = \ln k$。

这转化为函数 $g(x) = \ln x - x$ 与直线 $y = \ln k$ 的交点距离问题。

步骤二：利用切线不等式

$g(x)$ 是上凸函数。

- 在 $x=1$ 处切线为水平线 $y=-1$（极大值）。
- **关键技巧：** 分别在两个根附近的“特征点”做放缩。
  - 对于 $x_1$（小根），我们利用 $x_1 < 1$。
  - 对于 $x_2$（大根），我们利用 $x_2 > 1$。

利用经典放缩 $\ln x \le x-1$。

对于 $g(x) = \ln x - x$，代入放缩：



$$g(x) \le (x-1) - x = -1$$



这只是证明了极大值，不足以估计差值。我们需要更精细的放缩，例如泰勒展开的二阶形式或特定点切线。

进阶技巧：双切线“挤压”

若要证明 $|x_2 - x_1| > C$ (常数) 或类似范围，我们可以分别在 $x_1, x_2$ 处利用切线方程。

或者，利用 $e^x \ge ex$（过 $(1, e)$ 的切线）来处理原方程 $xe^{-x}=k \Rightarrow e^x = \frac{x}{k}$。

- 图解分析：

  画出 $y = e^x$ 和直线 $y = \frac{x}{k}$。

  直线过原点，斜率为 $1/k$。

  曲线 $y=e^x$ 下凸。

  切线放缩告诉我们，曲线被直线 $y = e(x-1) + e$ (在 $x=1$ 处切线) 支撑。

  通过计算直线 $y=x/k$ 与切线 $y=ex$ 或其他切线的交点，可以获得 $x_1, x_2$ 的内侧估计（Inner approximation）或外侧估计（Outer approximation）。

------

### 四、 综合应用：对数平均不等式与“偏移”

在处理 $x_1 + x_2$ 或 $x_2 - x_1$ 时，切线放缩的一个高级应用是推导**对数平均不等式**的几何形式。

对于 $f(x) = \ln x$，其为上凸函数。

对于任意 $x_1 \neq x_2$，割线斜率小于切线斜率（如果在中点处）或利用积分中值。

一个经典的切线放缩结论是：



$$\frac{x_1 - x_2}{\ln x_1 - \ln x_2} < \frac{x_1 + x_2}{2}$$



(对数平均数小于算术平均数)

**几何证明思路：**

1. 取梯形面积（割线围成的区域）与曲边梯形面积（积分）比较。

2. 或者，在 $x_0 = \frac{x_1+x_2}{2}$ 处作切线。由于 $\ln x$ 上凸，切线在曲线上方。

   

   $$\ln x \le \ln x_0 + \frac{1}{x_0}(x - x_0)$$

   

   将 $x_1, x_2$ 代入并求和，可以导出关于 $x_1+x_2$ 的强有力界限。

------

### 五、 结论与实战口诀

利用切线放缩解决交点横坐标问题，关键在于：

1. **定凸凹**：判断 $f(x)$ 的二阶导数符号，确定切线是在上方（上界）还是下方（下界）。
2. **找切点**：
   - 若估计 $x_1+x_2$，常取中点或极值点作为切点。
   - 若估计 $|x_1-x_2|$，常取区间端点或特殊整数点作为切点。
3. **化非线为线**：将 $f(x)=k$ 转化为 $L(x)=k$（$L$为切线），解出线性近似解 $\hat{x}$，以此作为 $x$ 的界。

**口诀：**

> 曲线弯弯难求根，切线直直好定分。
>
> 凸上切线盖头顶，凹下切线托足跟。
>
> 构造不等代原式，差值范围便成真。

### 建议后续练习


可以尝试使用此方法解决经典的**极值点偏移**问题，例如：“已知 $f(x) = x^2 - 2a\ln x$ 有两个零点 $x_1, x_2$，求证 $x_1 + x_2 > 2\sqrt{a}$”。这正是切线放缩（此处为利用 $x= \sqrt{a}$ 处的切线或展开）的典型应用场。
